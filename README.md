# Cartify 🛒

A modern, responsive e-commerce web application built with React and Tailwind CSS, featuring product browsing, smart filtering, cart management, and a seamless shopping experience.

> ⚠️ This project is frontend-only. No backend or real payment processing is implemented.

## ✨ Features

- 🏪 **Product Listing** — Browse products fetched dynamically from an API
- 🔍 **Search Functionality** — Quickly find products by name or keyword
- 🎛️ **Filtering & Sorting** — Enhance product discovery with filtering options
- 🛒 **Shopping Cart** — Add, remove, and manage products in the cart
- 🔐 **Authentication UI** — Login and Signup pages (UI only)
- 📱 **Fully Responsive Design** — Optimized for mobile, tablet, and desktop devices
- ⚡ **Fast Navigation** — Smooth page transitions using React Router
- 🔄 **Dynamic Data Fetching** — Real-time product data from Fake Store API

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| React.js | Component-based UI development |
| Tailwind CSS | Utility-first CSS framework |
| React Router | Client-side routing |
| Context API | Global state management |
| JavaScript (ES6+) | Application logic |
| Vite | Development and build tool |
| Fake Store API | Product data source |

---

## 🌍 API Used

This project uses the **Fake Store API** to fetch product data dynamically.

🔗 https://fakestoreapi.com/

The API provides:

- Product listings
- Product details
- Categories
- Pricing information
- Product images

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/CodeWithNouman99/Cartify-React_Project.git

# Navigate to the project folder
cd Cartify-React_Project

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open the application in your browser:

```text
http://localhost:5173
```
